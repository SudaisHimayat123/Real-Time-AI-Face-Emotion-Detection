# Outputs Directory

This directory stores all runtime-generated outputs:

- `videos/` — Recorded video files (MP4) with overlays
- `csv/` — Exported session emotion data (CSV)
- `sessions.db` — SQLite database with all session history
- `app.log` — Application log file
- `*_emotion_history_*.png` — Session emotion timeline graphs
- `*_emotion_pie_*.png` — Session emotion distribution pie charts

These files are generated automatically when you run the application.
This directory (except this README) is excluded from version control.
